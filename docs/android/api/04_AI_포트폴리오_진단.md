# AI 포트폴리오 진단 API

> 화면: `AiDiagnosisDialog.kt`, `SimulationViewModel.kt`

---

## 코드값 정의

### 투자 방식 (investmentType)

포트폴리오 시뮬레이션의 투자 방식을 구분합니다. AI 진단 시 투자 전략 분석에 활용됩니다.

| investmentType | 화면 표시명 | 설명 | AI 분석 관점 |
|----------------|------------|------|-------------|
| `LUMP_SUM` | 일시불 투자 | 목돈을 한 번에 투자 | 시장 타이밍, 변동성 노출 분석 |
| `REGULAR_SAVING` | 정액 적립식 | 매월 정해진 금액 투자 | 평균 매입단가, 분산 효과 분석 |

### LLM 모델 (llmModel)

AI 진단에 사용된 언어 모델 정보입니다. 응답에만 포함되며, 요청 시 지정할 수 없습니다.

| llmModel | 설명 | 특징 |
|----------|------|------|
| `claude-3-haiku-20240307` | Anthropic Claude 3 Haiku | 빠른 응답, 비용 효율적 (기본 모델) |

### 에러 코드

| 코드 | HTTP 상태 | 메시지 | 설명 | 대응 방법 |
|------|----------|--------|------|----------|
| `AI001` | 503 | AI 서비스 일시 불가 | LLM API 오류 또는 타임아웃 | 잠시 후 재시도 |
| `AI002` | 400 | 포트폴리오 데이터 오류 | ETF 비중 합계 ≠ 100% | 비중 합계 100% 확인 |
| `AI003` | 400 | 투자금액 범위 초과 | 금액이 1만원 ~ 100억원 범위 밖 | 범위 내 금액 입력 |
| `AI004` | 400 | ETF를 찾을 수 없습니다 | 존재하지 않는 ETF ticker | 올바른 티커 확인 |

### 키워드 (keywords)

AI가 분석한 포트폴리오 특성을 나타내는 태그입니다. 화면에 해시태그 형태로 표시됩니다.

| 키워드 예시 | 의미 | 표시 형태 |
|------------|------|----------|
| `기술주집중` | 기술 섹터 비중이 높음 | #기술주집중 |
| `고변동성` | 변동성이 높은 ETF 포함 | #고변동성 |
| `성장중심` | 성장주 ETF 위주 구성 | #성장중심 |
| `미국비중높음` | 미국 시장 ETF 비중 높음 | #미국비중높음 |
| `분산투자` | 여러 섹터에 분산됨 | #분산투자 |
| `배당중심` | 배당 ETF 위주 구성 | #배당중심 |
| `안정추구` | 채권/안정형 ETF 포함 | #안정추구 |
| `레버리지` | 레버리지 ETF 포함 | #레버리지 |

## POST /api/v1/ai/portfolio/review

포트폴리오 AI 진단 요청

### Request

```json
{
  "portfolio": {
    "totalAmount": 10000000,
    "investmentType": "LUMP_SUM",
    "etfs": [
      {
        "ticker": "091160",
        "name": "KODEX 반도체",
        "weight": 40
      },
      {
        "ticker": "305720",
        "name": "KODEX 2차전지산업",
        "weight": 35
      },
      {
        "ticker": "364690",
        "name": "KODEX 미국S&P500TR",
        "weight": 25
      }
    ]
  }
}
```

### Request 필드

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| portfolio.totalAmount | Long | Y | 총 투자금액 (1만원 ~ 100억원) |
| portfolio.investmentType | String | Y | 투자방식: `LUMP_SUM`(일시불) / `REGULAR_SAVING`(적립식) |
| portfolio.etfs | Array | Y | ETF 목록 |
| portfolio.etfs[].ticker | String | Y | ETF 티커 |
| portfolio.etfs[].name | String | Y | ETF 이름 |
| portfolio.etfs[].weight | Integer | Y | 비중 (%) - 합계 100% |

### Response

```
HTTP 201 Created
```

```json
{
  "success": true,
  "message": "Success",
  "code": "OK",
  "data": {
    "reviewId": 12345,
    "headline": "공격적인 수익 추구!",
    "subHeadline": "기술주 중심의 로켓 포트폴리오",
    "keywords": ["기술주집중", "고변동성", "성장중심", "미국비중높음"],
    "analysis": "이 포트폴리오는 반도체와 2차전지 등 성장 섹터에 집중되어 있습니다. 높은 수익을 기대할 수 있지만, 시장 변동성에 취약할 수 있습니다. 미국 S&P500 ETF가 25%로 분산 효과를 주고 있으나, 전체적으로 공격적인 성향입니다. 변동성을 줄이려면 채권형 ETF나 배당 ETF 추가를 고려해보세요.",
    "llmModel": "gpt-4o",
    "createdAt": "2025-03-10T14:30:00"
  },
  "timestamp": "2025-03-10T14:30:00"
}
```

### 화면 매핑 (AiDiagnosisDialog)

| API 필드 | 화면 표시 |
|----------|----------|
| headline | 메인 타이틀 (큰 글씨) |
| subHeadline | 서브 타이틀 (작은 글씨) |
| keywords | 해시태그 칩 (#기술주집중 등) |
| analysis | 요약 상세 박스 내 텍스트 |

### 화면 상태

| 상태 | 표시 |
|------|------|
| Loading | CircularProgressIndicator + "AI가 포트폴리오를 분석 중입니다..." |
| Success | 진단 결과 카드 |
| Error | "진단 결과를 불러오지 못했습니다.\n다시 시도해주세요." |
