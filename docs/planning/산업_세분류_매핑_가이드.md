# 산업 세분류 매핑 가이드

## 개요

ETF 구성종목의 산업 분류를 위해 **네이버 증권 테마** 데이터를 크롤링하여 세분류 코드를 매핑함.

---

## 분류 체계

| 레벨 | 명칭 | 예시 | 개수 | 용도 |
|------|------|------|------|------|
| Level 1 | 대분류 | C (제조업) | 3개 | KSIC 기준 |
| Level 2 | 중분류 | 26 (전자부품) | 4개 | KSIC 기준 |
| Level 3 | 소분류 | 261 (반도체 제조업) | 3개 | KSIC/WICS 기준 |
| **Level 4** | **세분류** | SEMI_HBM (HBM 반도체) | **62개** | **네이버 테마 기반 (커스텀)** |

---

## 세분류 매핑 프로세스

### 1단계: 네이버 테마 크롤링

**소스**: https://finance.naver.com/sise/theme.naver

```
수집 결과:
- 테마 수: 262개
- 고유 종목: 2,388개
```

크롤링 스크립트: `backend/data-service/scripts/crawl_company_data.py`

### 2단계: 테마 → 세분류 매핑

**매핑 파일**: `backend/data-service/data/crawled/theme_to_sector_mapping.json`

```json
{
  "mappings": {
    "HBM(고대역폭메모리)": {"code": "SEMI_HBM", "group": "IT_SEMI"},
    "반도체 대표주(생산)": {"code": "SEMI_MEM", "group": "IT_SEMI"},
    "시스템반도체": {"code": "SEMI_SYS", "group": "IT_SEMI"},
    "반도체 장비": {"code": "SEMI_EQP", "group": "IT_SEMI"},
    "2차전지": {"code": "BAT_CELL", "group": "ENERGY"},
    "2차전지(소재/부품)": {"code": "BAT_MAT", "group": "ENERGY"},
    "전기차": {"code": "AUTO_EV", "group": "AUTO"},
    ...
  }
}
```

### 3단계: 종목별 세분류 결정

한 종목이 여러 테마에 속할 경우 **우선순위** 적용:

1. 더 세부적인 테마 우선 (예: "2차전지(소재/부품)" > "2차전지")
2. 비중이 높은 테마 우선
3. 먼저 매칭된 테마 사용
