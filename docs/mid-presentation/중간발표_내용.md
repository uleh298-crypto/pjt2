## 발표 스크립트 요약

### 01 기획 배경 (슬라이드 3-6, 약 2분)
- ETF 시장 성장 현황 (1,000개+, 300조원)
- 기존 서비스 문제점 3가지
- 타겟: "1,000개 중 뭘 골라야 할지 모르는 사람들"
- 페르소나 3명 (입문자, 직장인, 전략적 투자자)
- 서비스 핵심: "탐색하고, 시뮬레이션하고, 추적하는"

### 02 요구사항 명세 (슬라이드 7-11, 약 3분)
- 핵심 기능 3가지: Discover / Simulate / Track
- 기능 명세 8개 (ETF 탐색, 상세, 포트폴리오, 시뮬레이션, 전략 저장/추적, 뉴스, 로그인)
- 기술 스택 (Spring Boot 3.2.2, FastAPI, PostgreSQL 16, Redis 7, RabbitMQ)
- ERD 36개 테이블
- 시스템 아키텍처 (Android → Spring Boot → FastAPI)

### 03 핵심 기능 (슬라이드 12-14, 약 3분)
- ETF 탐색: 클러스터 맵으로 섹터 분포 시각화
- 투자 시뮬레이션: 적립형/거치형, 수익률/MDD 백테스트
- 나의 전략: 포트폴리오 저장, 수익률 추적, 비교 분석
