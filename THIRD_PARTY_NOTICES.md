# Third-Party Notices

This repository includes third-party libraries and assets through Gradle and application resources. Before publishing a release or app binary, verify each dependency and asset license.

## Bundled Fonts

- Pretendard font files are bundled under `frontend/WYE/app/src/main/res/font/`.
- Keep the upstream font license notice with the project before publishing binaries or redistributing the font files.

## Removed Generated Data

Database dumps, crawled news text, generated CSV datasets, local service credentials, and remote maintenance scripts are intentionally excluded from the public source tree.
