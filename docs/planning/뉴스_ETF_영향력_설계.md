## 현재 뉴스 시스템

**네이버 증권 종목뉴스 방식**을 사용합니다:

| 항목 | 설명 |
|------|------|
| **데이터 소스** | 네이버 증권 종목뉴스 (이미 뉴스-종목 매핑됨) |
| **테이블** | `news_stock_mapping` (뉴스 ↔ 회사 매핑) |
| **ETF 관련 뉴스** | ETF 구성종목(`etf_stock_composition`)의 뉴스 조회 |
| **LLM 분석** | 키워드/요약만 생성 (`news_article.keywords`, `content_summary`) |

## 폐기된 시스템 요약 (참고용)

기존에 계획했던 시스템:

1. **news_etf_influence 테이블**: 뉴스 → ETF 직접 매핑 (LLM 분석 기반)
2. **영향도 검증**: 장 마감 후 실제 ETF 주가 변동률로 검증
3. **타임라인**: ETF 상세 페이지에 뉴스 이슈 타임라인 표시

**폐기 사유**: 네이버가 이미 뉴스-종목 매핑을 제공하므로 LLM 분석이 불필요.
