# 알림 API

> 화면: `NotificationScreen.kt`, `NotificationSettingsScreen.kt`

## 코드값 정의

### 카테고리 (category)

알림을 분류하는 최상위 카테고리입니다. 알림 목록 조회 시 필터링에 사용됩니다.

| category | 설명 | 용도 |
|----------|------|------|
| `ETF` | ETF 관련 알림 | ETF 상장/폐지 등 ETF 자체 이벤트 |
| `PORTFOLIO` | 포트폴리오 관련 알림 | 내 포트폴리오의 수익률, 리밸런싱 등 |
| `NEWS` | 뉴스 관련 알림 | 관심 ETF 관련 뉴스 발생 시 |
| `SYSTEM` | 시스템 공지 알림 | 서비스 공지사항, 점검 안내 등 |

### 알림 유형 코드 (alertTypeCode)

실제 알림의 세부 유형을 구분하는 코드입니다. DB의 `alert_type` 테이블에 정의되어 있습니다.

#### SYSTEM 카테고리

| alertTypeCode | 설명 | 발생 시점 | 메시지 예시 |
|---------------|------|----------|------------|
| `SYSTEM_ANNOUNCEMENT` | 시스템 공지사항 | 운영자가 공지 등록 시 | "서비스 점검 안내" |

#### ETF 카테고리

| alertTypeCode | 설명 | 발생 시점 | 메시지 예시 |
|---------------|------|----------|------------|
| `ETF_LISTING` | ETF 신규 상장 | 새로운 ETF 상장 시 | "KODEX 2차전지 ETF가 상장되었습니다" |
| `ETF_DELISTING_SCHEDULED` | ETF 상장폐지 예정 | 관심 ETF 폐지 예정 공시 시 | "TIGER 반도체 ETF가 2025-04-01 상장폐지 예정입니다" |
| `ETF_DELISTING_COMPLETED` | ETF 상장폐지 완료 | 관심 ETF 실제 폐지 시 | "TIGER 반도체 ETF가 상장폐지 되었습니다" |

#### PORTFOLIO 카테고리

| alertTypeCode | 설명 | 발생 시점 | 메시지 예시 |
|---------------|------|----------|------------|
| `PORTFOLIO_REBALANCING_SCHEDULED` | 리밸런싱 예정 | 포트폴리오에 포함된 ETF 리밸런싱 예정 시 | "'내 포트폴리오' 포함 ETF 'KODEX 200' 리밸런싱이 2025-03-15에 예정되어 있습니다" |
| `PORTFOLIO_REBALANCING_COMPLETED` | 리밸런싱 완료 | 포트폴리오에 포함된 ETF 리밸런싱 완료 시 | "'내 포트폴리오' 포함 ETF 'KODEX 200' 리밸런싱이 완료되었습니다" |
| `PORTFOLIO_RETURN_5PCT` | 수익률 +5% 도달 | 포트폴리오 수익률이 +5% 달성 시 | "'기술주 집중' 포트폴리오 수익률이 +5%를 달성했습니다!" |
| `PORTFOLIO_RETURN_10PCT` | 수익률 +10% 도달 | 포트폴리오 수익률이 +10% 달성 시 | "'기술주 집중' 포트폴리오 수익률이 +10%를 달성했습니다!" |
| `PORTFOLIO_LOSS_5PCT` | 손실률 -5% 도달 | 포트폴리오 손실률이 -5% 도달 시 | "'기술주 집중' 포트폴리오 손실률이 -5%에 도달했습니다" |
| `PORTFOLIO_LOSS_10PCT` | 손실률 -10% 도달 | 포트폴리오 손실률이 -10% 도달 시 | "'기술주 집중' 포트폴리오 손실률이 -10%에 도달했습니다" |

#### NEWS 카테고리

| alertTypeCode | 설명 | 발생 시점 | 메시지 예시 |
|---------------|------|----------|------------|
| `NEWS_ETF_RELATED` | 관심 ETF 관련 뉴스 | 관심 등록한 ETF 관련 뉴스 수집 시 | "관심 ETF 'KODEX 반도체' 관련 뉴스: 반도체 업황 회복세" |

### 설정 그룹 (settingGroup)

사용자가 알림 설정 화면에서 ON/OFF 할 수 있는 단위입니다. 하나의 설정 그룹에 여러 alertTypeCode가 포함될 수 있습니다.

| settingGroup | 화면 표시명 | 설명 | 포함된 alertTypeCode |
|--------------|------------|------|----------------------|
| `APP_NOTIFICATION` | 앱 알림 | 서비스 공지사항, 업데이트 안내 등 | SYSTEM_ANNOUNCEMENT |
| `ETF_LISTING` | ETF 상장 알림 | 새로운 ETF 상장 소식 | ETF_LISTING |
| `ETF_DELISTING` | ETF 상장 폐지 알림 | 관심 ETF의 상장폐지 예정/완료 | ETF_DELISTING_SCHEDULED, ETF_DELISTING_COMPLETED |
| `PORTFOLIO_REBALANCING` | 포트폴리오 리밸런싱 알림 | 포트폴리오 내 ETF 구성종목 변경 | PORTFOLIO_REBALANCING_SCHEDULED, PORTFOLIO_REBALANCING_COMPLETED |
| `PORTFOLIO_RETURN` | 포트폴리오 수익률 알림 | 포트폴리오 수익률/손실률 임계치 도달 | PORTFOLIO_RETURN_5PCT, PORTFOLIO_RETURN_10PCT, PORTFOLIO_LOSS_5PCT, PORTFOLIO_LOSS_10PCT |
| `NEWS_NOTIFICATION` | 뉴스 수신 알림 | 관심 ETF 관련 뉴스 알림 | NEWS_ETF_RELATED |

### 참조 타입 (referenceType)

알림 클릭 시 이동할 화면을 결정하는 타입입니다.

| referenceType | 이동 화면 | 라우트 | 사용 필드 | 설명 |
|---------------|----------|--------|----------|------|
| `ETF` | ETF 상세 화면 | `/etf/{referenceTicker}` | `referenceTicker` | ETF 종목코드(티커) 사용 |
| `PORTFOLIO` | 포트폴리오 상세 화면 | `/portfolio/{referenceId}` | `referenceId` | portfolio.id 사용 |
| `NEWS` | 뉴스 상세 화면 | `/news/{referenceId}` | `referenceId` | news_article.id 사용 |
| `null` | 이동 없음 | - | - | 시스템 공지 등 이동할 화면 없음 |

> **중요**: ETF의 경우 `referenceId`(숫자 ID) 대신 `referenceTicker`(종목코드)를 사용합니다.
> ETF API가 `GET /api/v1/etf/{ticker}` 형식이기 때문입니다.

## GET /api/v1/alerts

최근 7일 알림 목록 조회

### Request

| 파라미터 | 타입 | 필수 | 설명 |
|----------|------|------|------|
| category | String | N | 카테고리 필터 (all/ETF/PORTFOLIO/NEWS/SYSTEM) |

### Response

```json
{
  "success": true,
  "data": {
    "alerts": [
      {
        "id": 123,
        "alertTypeCode": "ETF_LISTING",
        "alertTypeName": "ETF 신규 상장",
        "category": "ETF",
        "referenceType": "ETF",
        "referenceId": 45,
        "referenceTicker": "069500",
        "title": "KODEX 반도체 신규 상장",
        "message": "KODEX 반도체 ETF가 상장되었습니다.",
        "isRead": false,
        "createdAt": "2025-03-10T14:30:00"
      }
    ],
    "unreadCount": 12
  }
}
```

### 응답 필드 설명

| 필드 | 타입 | 설명 |
|------|------|------|
| id | Long | 알림 ID |
| alertTypeCode | String | 알림 유형 코드 |
| alertTypeName | String | 알림 유형 이름 |
| category | String | 카테고리 (ETF/PORTFOLIO/NEWS/SYSTEM) |
| referenceType | String | 참조 대상 유형 (ETF/PORTFOLIO/NEWS/null) |
| referenceId | Long | 참조 대상 ID (PORTFOLIO, NEWS에서 사용) |
| referenceTicker | String | ETF 종목코드 (referenceType=ETF일 때만 존재) |
| title | String | 알림 제목 |
| message | String | 알림 메시지 |
| isRead | Boolean | 읽음 여부 |
| createdAt | String | 생성일시 (ISO 8601) |
